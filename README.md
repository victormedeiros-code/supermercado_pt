# Sistema de Gestão de Stock - Supermercado PT

Sistema de terminal desenvolvido em **Python** para gestão de inventário, utilizando bases de dados relacionais e interface rica no console.

## Funcionalidades
* **Listagem de Produtos:** Exibição de stock em tabelas coloridas e formatadas.
* **Atualização Dinâmica:** Alteração de quantidades diretamente no banco de dados.
* **Interface no Terminal:** Menu interativo com validação de dados e limpeza de ecrã.

## Tecnologias Utilizadas
* **Python 3.x**
* **SQLite3:** Para persistência de dados.
* **Rich Library:** Para estilização de tabelas, painéis e prompts.
* **OS Module:** Para manipulação de comandos de sistema (Windows/Linux).

## Como rodar o projeto

1. **Clone o repositório:**
   `git clone [URL_DO_REPOSITÓRIO]`

2. **Crie um ambiente virtual:**
   `python -m venv venv`

3. **Ative o venv:**
   * No Windows: `.\venv\Scripts\activate`
   * No Linux/Mac: `source venv/bin/activate`

4. **Instale as dependências:**
   `pip install -r requirements.txt`

5. **Execute o programa:**
   `python main.py`
